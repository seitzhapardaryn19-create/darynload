# Telegram-бот для скачивания видео и музыки

Бот скачивает видео (MP4) и аудио (MP3) из **YouTube, TikTok и Instagram** по ссылке.

Стек: Python 3.12, aiogram 3, yt-dlp, ffmpeg, Docker.

## Как это работает

1. Пользователь отправляет боту ссылку
2. Бот предлагает выбрать формат: Видео (MP4) или Аудио (MP3)
3. yt-dlp скачивает файл, бот отправляет его в чат
4. Временные файлы удаляются

Ограничение Telegram Bot API: бот не может отправить файл больше **50 МБ**.

---

## Шаг 1. Создать бота в Telegram

1. Откройте в Telegram бота **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя и username (должен заканчиваться на `bot`)
4. Скопируйте **токен** — он понадобится дальше

## Шаг 2. Запустить локально (для проверки)

Нужен установленный Python 3.10+ и ffmpeg.

```bash
cd telegram-bot
pip install -r requirements.txt

# Windows (PowerShell):
$env:BOT_TOKEN="ваш_токен"
# Linux / macOS:
export BOT_TOKEN="ваш_токен"

python bot.py
```

Отправьте боту ссылку на YouTube-видео — должно работать.

## Шаг 3. Бесплатный деплой 24/7 на Render

1. Загрузите папку `telegram-bot` в репозиторий на **GitHub**
2. Зарегистрируйтесь на [render.com](https://render.com) (через GitHub)
3. Нажмите **New → Web Service** и выберите ваш репозиторий
4. Настройки:
   - **Root Directory**: `telegram-bot` (если бот в подпапке)
   - **Runtime**: Docker (Render сам найдёт Dockerfile)
   - **Instance Type**: Free
5. В разделе **Environment** добавьте переменную:
   - `BOT_TOKEN` = ваш токен от BotFather
6. Нажмите **Deploy**

### Важно: не дать боту "уснуть"

Бесплатный тариф Render усыпляет сервис после 15 минут без HTTP-запросов.
В коде уже есть health-check эндпоинт (`GET /`), нужно только пинговать его:

1. Скопируйте URL вашего сервиса на Render (вида `https://ваш-бот.onrender.com`)
2. Зарегистрируйтесь на [uptimerobot.com](https://uptimerobot.com) (бесплатно)
3. Создайте монитор: **HTTP(s)**, ваш URL, интервал **5 минут**

Готово — бот работает 24/7 бесплатно.

### Альтернативы Render

## Известные ограничения

- **50 МБ** — лимит Telegram для ботов. Длинные видео в высоком качестве не отправятся (код автоматически ограничивает качество до 720p).
- **YouTube может блокировать** запросы с IP датацентров (Render и т.п.). TikTok и Instagram обычно работают стабильнее. Это известная особенность всех подобных ботов.
- Приватные видео и Instagram-посты с закрытых аккаунтов скачать нельзя.
